Meu Portfólio

Site de Comércio Eletrônico

Desenvolvi um site de comércio eletrônico completo, com funcionalidades de carrinho de compras, processamento de pagamentos e sistema de gerenciamento de produtos. Usei HTML, CSS, JavaScript, e PHP para a implementação, e MySQL para o banco de dados.

Aplicativo de Lista de Tarefas

Criei um aplicativo de lista de tarefas para ajudar os usuários a organizar suas atividades diárias. O aplicativo permite adicionar, editar e excluir tarefas, além de marcar tarefas como concluídas. Utilizei React para a interface e Node.js para o backend.

Blog Pessoal

Desenvolvi um blog pessoal com um sistema de gerenciamento de conteúdo (CMS) personalizado, onde os usuários podem criar e editar posts, adicionar comentários, e gerenciar categorias. A interface foi criada com HTML, CSS, e JavaScript, enquanto o backend foi feito com Django.

Dashboard de Análise de Dados

Criei um dashboard interativo para análise de dados financeiros, utilizando gráficos e tabelas dinâmicas. O projeto foi construído com Python e Flask, e os dados são visualizados com a biblioteca Plotly. A interface é responsiva e se adapta a diferentes dispositivos.
